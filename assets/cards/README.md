# Tarot card image assets

Put your finished card-face artwork here. The site already looks for these paths:

- Major Arcana: `assets/cards/major/<english-slug>.png`, for example `assets/cards/major/fool.png`
- Minor Arcana: `assets/cards/minor/<suit>-<rank>.png`, for example `assets/cards/minor/swords-ace.png`

If an image file is missing, the card automatically falls back to the generated mandala artwork, so you can upload the real card images gradually.
